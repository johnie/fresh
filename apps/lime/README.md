# Lime

TanStack Start app starter.

## Install

Copy this app out of the repo with `gitpick`:

```sh
npx gitpick johnie/fresh/apps/lime my-lime-app
```

## Run locally

```sh
cd my-lime-app
nvm use
pnpm install
pnpm dev
```

## Build

```sh
pnpm build
```

## Test

Run the test suite once:

```sh
pnpm test
```

Run Vitest in watch mode:

```sh
pnpm test:watch
```

## Check

Run Ultracite checks:

```sh
pnpm check
```
